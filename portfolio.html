BreadCrumbs

BreadCrumbs is a hide and seek team project involing users who are geographically close to each other. This application uses Google Firebase's authentication and Google map's API. Also using the Cloudinary API to allow users to upload pitures as clues to their location. Built with HTML, CSS, JavaScript, AJAX, and Node.js.
